# 🚀 AI-Generated Interview Tool

An intelligent AI-powered platform that helps users prepare for interviews by analyzing their resume, job description, and personal inputs to generate tailored interview questions, ATS scores, and optimized resumes.

---

## 📌 Overview

The **AI-Generated Interview Tool** is designed to simulate real-world interview preparation. It enables users to:

- Register and login securely
- Upload or input their resume
- Provide job descriptions and personal details
- Get AI-generated **technical + behavioral questions**
- Receive an **ATS (Applicant Tracking System) score**
- Optimize resumes for better hiring chances
- Download ATS-friendly resumes
